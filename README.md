# Smart Adaptive Study Planner

An intelligent, adaptive study scheduling engine built for the Hackathon MVP.

## Features
-   **Priority-based Scheduling**: Automatically prioritizes subjects based on difficulty and exam urgency.
-   **Visual Dashboard**: See your daily study plan at a glance.
-   **Premium UI**: Modern dark theme with glassmorphism.
-   **Offline Capable**: Works entirely in the browser with Local Storage.

## tech Stack
-   React + TypeScript + Vite
-   Tailwind CSS
-   Lucide React

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```
